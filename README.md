# 自定义连点器
自己vibe coding了一个小工具,做着玩玩的,如果有类似的需求可以下载试试.觉得好用可以给个star
Windows 轻量化桌面自动化点击工具 — 高性能、零臃肿、纯绿色、单文件免安装。

## 技术栈

**Rust** + **egui** (UI) + **Win32 原生 API** (键鼠驱动)

编译输出为单个独立 EXE，百 KB 级体积，常驻内存 10MB 以内，闲置几乎零资源占用。

## 核心功能

### 模块一：绑定按键点击

将任意键盘按键绑定到屏幕固定坐标，按下按键即可触发点击。

- **F8** 一键拾取当前鼠标坐标
- 手动输入 X/Y 绝对坐标，自动校验合法性
- 支持两种触发模式：
  - **单击模式**：按一次，点一次
  - **连点模式**：按一次启动 / 停止高频连点，间隔最低 1ms
- **多方案管理**：可创建、切换、重命名、删除多组按键绑定方案
- **坐标收藏**：常用坐标一键收藏，随时调用
- 全局热键：**F9** 启用按键监听，**F10** 停止所有操作
- 点击通过 `SendInput` 绝对坐标注入，不抢夺鼠标光标

### 模块二：脚本连点

基于自定义 JSON 脚本格式，兼顾新手录制与高阶手写。

- **录制脚本**（F11 切换）：底层静默录制鼠标移动、左键、右键、操作间隔，自动生成标准 JSON 脚本
- **手写脚本**：内置编辑器，支持手动编写点击坐标、类型、延迟、循环逻辑，与录制脚本格式完全互通
- **脚本回放**：支持播放、停止、循环（无限 / 指定次数）、速度无级调节（0.1x ~ 10x）
- **脚本管理**：本地保存、重命名、删除、导入、导出 JSON 文件
- **语法校验**：内置校验器，精准提示错误位置

#### JSON 脚本格式示例

```json
{
  "name": "示例脚本",
  "version": 1,
  "actions": [
    { "type": "move",  "x": 500, "y": 300, "delay_ms": 100 },
    { "type": "click", "x": 500, "y": 300, "delay_ms": 200, "absolute": true },
    { "type": "right_click", "x": 600, "y": 400, "delay_ms": 500 },
    { "type": "delay", "delay_ms": 1000 }
  ],
  "loop": { "enabled": false, "count": 0 },
  "speed_multiplier": 1.0
}
```

## 热键一览

| 热键 | 功能 |
|------|------|
| F8 | 拾取当前鼠标坐标 |
| F9 | 启用按键绑定监听 / 脚本播放 |
| F10 | 停止所有操作（连点、播放、录制） |
| F11 | 切换录制状态（开始 / 停止录制） |

窗口最小化后热键仍然有效。

## 运行 / 编译

### 直接运行

下载 `自定义连点器.exe` 双击启动，绿色免安装。

### 从源码编译

```bash
# 需要 Rust 1.70+
cargo build --release
```

编译配置已优化：LTO 瘦身 + 符号裁剪 + 静态编译，输出单文件 EXE。

## 项目结构

```
src/
├── main.rs              # 入口，字体加载，事件循环
├── app.rs               # 应用状态与业务逻辑
├── core/
│   ├── clicker.rs       # Win32 鼠标点击注入（绝对坐标/无抢夺）
│   ├── hotkey.rs        # 全局键盘钩子（WH_KEYBOARD_LL）
│   └── recorder.rs      # 全局鼠标钩子录制（WH_MOUSE_LL）
├── script/
│   ├── types.rs         # JSON 脚本类型定义
│   ├── parser.rs        # 脚本解析与校验
│   ├── player.rs        # 脚本回放引擎
│   └── storage.rs       # 脚本文件存取（CRUD）
├── ui/
│   ├── main_window.rs   # 主窗口布局（标签栏 + 状态栏）
│   ├── click_mode.rs    # 绑定按键点击页面
│   └── script_mode.rs   # 脚本连点页面
└── utils/
    ├── bindings.rs      # 按键绑定数据结构与 VK 码映射
    ├── coordinate.rs    # 屏幕坐标工具
    ├── dialog.rs        # 文件对话框
    └── favorites.rs     # 坐标收藏
```

## 系统要求

- Windows 10 / Windows 11
- 无需安装任何运行环境

## 设计原则

- 无广告、无弹窗、无捆绑、无后台驻留
- 关闭即销毁所有进程，无注册表残留
- 不写入注册表，配置文件与 EXE 同目录
